## Token Naming — Doubled-Prefix Audit

Rule: `@theme inline` with `--color-X` generates utility `[prefix]-X`. The color name portion must NOT start with a CSS property prefix or it doubles.

### Names that were caught and renamed:

| Original (would break)         | Problem                        | Fixed name              | Generated class          |
|-------------------------------|-------------------------------|------------------------|-------------------------|
| `--color-panel-border`        | → `border-panel-border`        | `--color-line-panel`   | `border-line-panel` ✅  |
| `--color-input-bg`            | → `bg-input-bg` (clunky)      | `--color-input-surface`| `bg-input-surface` ✅   |
| `--color-input-border`        | → `border-input-border`        | `--color-input-edge`   | `border-input-edge` ✅  |
| `--color-input-border-focus`  | → `border-input-border-focus`  | `--color-input-edge-focus` | `border-input-edge-focus` ✅ |
| `--color-input-text`          | → `text-input-text`            | `--color-input-value`  | `text-input-value` ✅   |

## Depth Model

**Warm Stone (default):** Sidebar is RAISED — `surface-panel (#231f1d)` is lighter than `surface-base (#1c1917)`. Active tab blends into sidebar.

**Zed Dark (alt):** Sidebar is RECESSED — `surface-panel (#21252b)` is darker than `surface-base (#282c34)`. Active tab lifts to editor depth.

Both themes: Inputs are RAISED — `input-surface` = `surface-raised` level.

## Migration Maps

### Flagged Classes → Tokens

| Flagged class                    | New token class              | Files affected                                      |
|---------------------------------|-----------------------------|----------------------------------------------------|
| `text-neutral-700`              | `text-content-disabled`      | SidebarDecksTab, SampleHandModal, WorkspaceToolbar, VisualCard, ListCardTable |
| `bg-neutral-950`                | `bg-surface-deep`            | SearchBar                                           |
| `hover:bg-neutral-600`          | `hover:bg-surface-hover`     | VisualCard (qty buttons)                            |
| `border-neutral-600`            | `border-line-hover`          | CategoryChips, SearchBar (freeform badge)           |
| `border-neutral-500`            | `border-line-hover`          | Sidebar (sort direction hover)                      |
| `focus-within:border-neutral-600` | `focus-within:border-input-edge-focus` | SearchBar                                |

### Sidebar Panel Migration

| Component               | Current class       | New class               |
|-------------------------|--------------------|-----------------------|
| Sidebar.tsx (shell bg)  | `bg-surface-base`  | `bg-surface-panel`     |
| SidebarRail.tsx (bg)    | `bg-surface-base`  | `bg-surface-panel`     |
| SidebarDecksTab.tsx (bg)| `bg-surface-base`  | `bg-surface-panel`     |
| Sidebar border-right    | `border-line-subtle` | `border-line-panel`  |
| Active sidebar tab bg   | `bg-surface-raised`| `bg-surface-panel`     |
| Sidebar hover rows      | `hover:bg-surface-raised` | `hover:bg-surface-panel-raised` |

### Input Token Migration

| Element                     | Current                              | New                                    |
|----------------------------|--------------------------------------|----------------------------------------|
| SearchBar input bg         | `bg-surface-raised` or `bg-neutral-950` | `bg-input-surface` (container stays `bg-surface-deep`) |
| SearchBar input text       | `text-content-heading`               | `text-input-value`                     |
| SearchBar placeholder      | `text-content-faint`                 | `text-input-placeholder`               |
| SearchBar focus ring       | `focus-within:border-neutral-600`    | `focus-within:border-input-edge-focus` |
| FilterPanel text inputs    | `bg-surface-raised`                  | `bg-input-surface`                     |
| FilterPanel input borders  | `border-line-default`                | `border-input-edge`                    |
| WorkspaceToolbar name hover| `border-line-default`                | `border-input-edge`                    |
| ImportModal textarea       | `bg-surface-raised`                  | `bg-input-surface border-input-edge`   |

### Exclusions (do NOT migrate)
- Opacity variants (e.g. `bg-neutral-800/50`, `border-neutral-500/50`, `border-neutral-700/50`)
- Accent colors (blue, red, green, yellow, amber)
- `text-neutral-100` (flagged — no token yet, stays raw)
